# 🍔 Snacking Template

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)

**Snacking Template** est une solution moderne et prête à l'emploi pour créer rapidement des interfaces de restauration rapide, de click-and-collect ou de menus digitaux. Ce template met l'accent sur la performance et l'expérience utilisateur mobile.

[🎥 Voir la Démo](#) | [🐞 Signaler un Bug](https://github.com/Sulayman74/snacking_template/issues) | [💡 Proposer une Feature](https://github.com/Sulayman74/snacking_template/issues)

---

## ✨ Fonctionnalités

- 📱 **Mobile First** : Interface optimisée pour les smartphones.
- ⚡ **Performance** : Chargement rapide des images et des composants.
- 🛒 **Panier Dynamique** : Gestion fluide de l'ajout/suppression d'articles.
- 🎨 **Customisable** : Couleurs et polices facilement modifiables via des variables CSS/Thèmes.
- 📂 **Architecture Propre** : Code structuré pour être facilement maintenu et étendu.

## 🛠 Stack Technique

Voici les technologies principales utilisées dans ce projet :

* **Frontend :** [Vanilla js]
* **Styling :** [Tailwind CSS](https://tailwindcss.com/)
* **Icons :** [Lucide React](https://lucide.dev/) / FontAwesome
